### 5.6 Executed Proof Route (full)
We use the CPU-only surrogate
`n_eff^surr(d, n_win, b) = n_win * (1 - 2d)^2 / (1 + b / 200)`
clipped below by `1`, and fit the screening quantities
`gamma_CI = 0.015 + 18 / n_eff^surr + 0.06 (d - 0.35) + 0.002 log_2(b)`
and
`Delta_FR = min(0.20, 0.01 + 0.8 gamma_CI + 0.005 (20 / b))`.

Define
`Gamma_safe = {(d, n_win, b): Delta_FR(d, n_win, b) <= B_FR}`
with `B_FR = 0.15`.

Derivation summary:
- For fixed `(n_win, b)`, `n_eff^surr` decreases as `d` increases, so `18 / n_eff^surr` increases.
- The explicit `0.06 (d - 0.35)` term further increases `gamma_CI` with `d`.
- Hence the fitted envelope is monotone in `d`, conditional on `A53`.
- The screen `Delta_FR <= B_FR` therefore shrinks as `d` approaches `0.45`.
- On today’s grid, the candidate safe subset is:
  - `d = 0.35`, `n_win = 4000`, `b in {10, 20, 40, 80}`;
  - `d = 0.40`, `n_win = 4000`, `b in {10, 20, 40}`;
  - no tested point at `d = 0.45`.

Selected grid evidence:
